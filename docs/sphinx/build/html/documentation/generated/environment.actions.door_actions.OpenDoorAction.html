is_possible(self, grid_world, agent_id, **kwargs)¶

Checks whether the action is possible in the world when performed by the given agent.

Parameters

  • grid_world – A pointer to the actual world object.

  • agent_id – The id known in the grid world as an agent that peforms this action.

Returns

Returns two things; a boolean signalling whether the action is possible or not, and a string telling

you why the action was not possible.

mutate(self, grid_world, agent_id, **kwargs)¶

Performs the actual action in the world. Opens the door, by changing the colour and setting the open_door property

Parameters

  • grid_world – A pointer to the actual world object.

  • agent_id – The id known in the grid world as an agent that peforms this action.

  • kwargs – Requires the door_range (int indicating cells range) and object_id arguments.

Returns

An action result depicting the action’s success or failure and reason/description of that result.
